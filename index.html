xAPI Launch

An algorithm for launching xAPI learning experience.
Learn More


info

No hard coded configuration in content

xAPI Launch enables ISD/developers to track learning experiences from any resource without some out-of-band method to add LRS credentials to the content, and without asking the user to input these credentials into some untrused third-party system.

perm_media

Support multiple content types

xAPI Launch is agnostic in the type of content it launches. This allows ISD/developers to use whatever media type they determine is most effective for their learning goals, such as hosted content, native mobile applications, or simulators.
